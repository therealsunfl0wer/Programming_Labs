# Композиція

1. Напишіть функцію `pipe`, яка компонує передані до неї аргументи зліва
   направо. `const pipe = (...fns) => x => ...` Якщо хоча б один із аргументів не
   буде функцією, `pipe` має кинути помилку. Наприклад, якщо є три функції:

```js
const inc = (x) => ++x;
const twice = (x) => x * 2;
const cube = (x) => x ** 3;
```

І нам потрібно скомпонувати їх так `const f = pipe(inc, twice, cube);` то при
виклику `const x = f(5);` потрібно очікувати, що `x` прийме значення `1728`.
А якщо скомпонувати `const f = pipe(inc, inc);` то при виклику
`const x = f(7);` значення `x` буде `9`. Але якщо передати не-функцію в
`pipe`, наприклад: `const f = pipe(inc, 7, cube);` то, не чекаючи виклику `f`,
одразу отримаємо помилку.

2. Реалізуйте композицію функцій справа наліво (без рекурсії), яка буде
   приглушувати помилки: якщо складена функція кине помилку, виконання завершується
   з `undefined`, і на помилки можна підписатися через `f.on('error', e => { ... });`.
