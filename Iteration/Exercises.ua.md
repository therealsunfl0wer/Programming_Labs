# Вправи

## Ітерування циклами

Реалізуйте функцію `sum(...args)`, яка сумує всі свої аргументи п'ятьма
різними способами. Приклади викликів з результатами:

```js
const a = sum(1, 2, 3); // a === 6
const b = sum(0); // b === 0
const c = sum(); // c === 0
const d = sum(1, -1, 1); // d === 1
const e = sum(10, -1, -1, -1); // e === 7
```

1. Цикл `for`
2. Цикл `for..of`
3. Цикл `while`
4. Цикл `do..while`
5. Метод `Array.prototype.reduce()`

## Ітерування по двовимірному масиву

6. Знайдіть максимальний елемент в двовимірному масиві

```js
const m = max([
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9],
]);
console.log(m); // 9
```
