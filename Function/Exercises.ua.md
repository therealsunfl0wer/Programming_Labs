# Вправи

## Функції

1. Реалізуйте функцію `random(min, max)`, що повертає псевдовипадкове значення
   від `min` до `max`. Використайте `Math.random()` та `Math.floor()`. При виклику
   `random(max)` вважайте `min = 0`.
2. Реалізуйте функцію `generateKey(length, characters)`, що повертає рядок
   випадкових символів з набору `characters` довжини `length`. Наприклад:

```js
const characters = "abcdefghijklmnopqrstuvwxyz0123456789";
const key = generateKey(16, characters);
console.log(key); // eg599gb60q926j8i
```

3. Перетворіть IPv4 адресу в число, що містить 4 байти адреси з бітовим
   зсувом у такий спосіб:

- 1 байт зсунутий на 3 \* 8 біт вліво
- 2 байт зсунутий на 2 \* 8 біт вліво
- 3 байт зсунутий на 1 \* 8 біт вліво
- 4 байт не зсунутий
  Приклад: '10.0.0.1' -> отримуємо [10,0,0,1] -> зсуваємо і сумуємо -> 167772161
  Оптимізуйте код з використанням `Array.prototype.reduce`. Значення за
  замовчуванням: '127.0.0.1'.

4. Реалізуйте інспекцію об'єкта:

- Проітеруйте всі ключі об'єкта `iface`
- Візьміть ключи функціонального типу
- Для кожної функції отримайте кількість аргументів
- Збережіть результати в двовимірний масив

Наприклад, з:

```js
{
   m1: x => [x],
   m2: function (x, y) {
      return [x, y];
   },
   m3(x, y, z) {
      return [x, y, z];
   }
}
```

отримаємо:

```js
[
  ["m1", 1],
  ["m2", 2],
  ["m3", 3],
];
```
